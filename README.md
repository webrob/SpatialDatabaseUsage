SpatialDatabaseUsage
====================

This project shows usage of spatial database. It takes advantage of the following technologies:
<ul>
    <li> JDBC </li>
    <li> MySQL with spatial extension </li>
    <li> JAX-B </li>
    <li> JAX-RS </li>
    <li> CDI </li>
    <li> JSON </li>
    <li> jQuery </li>
</ul>
